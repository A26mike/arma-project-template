# Arma3 AddOn Project template

A project template for Arma3 AddOn projects using the ACE3 framework.


### Usage

Search and replace all mentions of the following:
```
proj_templ by {your_prefix}
```

```
PROJ_TEMPL by {PROJ_TEMPL}
```

##### Example:

```
proj_templ by acex
```

```
PROJ_TEMPL by ACEX
```
